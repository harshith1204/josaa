### Fee Structure (B.Tech 2025 Batch)

**1st Semester (Indian): Rs 1,64,750**

| Component | Amount |
|-----------|--------|
| Campus Amenities (one-time, refundable) | Rs 20,000 |
| Hostel (one-time, refundable) | Rs 5,000 |
| Academic | Rs 10,750 |
| Tuition | Rs 1,00,000 |
| Hostel | Rs 6,500 |
| Other Institute | Rs 14,000 |
| Gymkhana | Rs 5,000 |
| Transport | Rs 1,500 |
| Health Insurance | Rs 2,000 |

- **International Students:** Rs 3,64,750 (1st semester)
- **2nd Semester Onwards (Indian):** Rs 1,25,000
- **SC/ST/PH:** Tuition fee exempted
- **Family income below 1 lakh:** Full fee remission
- **Family income 1-5 lakh:** 2/3rd fee remission

### Mess
- **Assignment:** Each hostel has its own separate mess
- **1st Years:** Combined in CV Raman mess

### WiFi & Internet
- WiFi available in all hostels
- LAN in CV Raman and Aryabhatta hostels
- Kalam hostel has WiFi only (no LAN)
- 5G available
- Poor mobile network in academic departments but WiFi available there

### Attendance
- 75% mandatory per semester per course
- Below 75% = not eligible for exam (must do summer term)
- Valid medical reasons accepted
